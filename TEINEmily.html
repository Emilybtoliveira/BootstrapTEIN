GUIA BOOTSTRAP

O Bootstrap é uma ferramenta gratuita para desenvolvimento HTML, CSS e JS. O framework tem o objetivo de criar projetos responsivos para dispositivos móveis na web. Possui uma diversidade de componentes em JavaScript (jQuery) para serem implementados apenas acrescentando algumas configurações no código, sem a necessidade de criar scripts e mais scripts.

Adicionando o Bootstrap a um projeto

Para adicionar o Bootstrap rapidamente a um projeto basta adicionar algumas tags no seu arquivo HTML.

Tag responsiva (Dentro da tag HEAD):

‹meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no"›

CSS (Dentro da tag HEAD):

‹link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css" integrity="sha384-MCw98/SFnGE8fJT3GXwEOngsV7Zt27NXFoaoApmYm81iuXoPkFOJwJ8ERdknLPMO" crossorigin="anonymous"›

JS (Dentro da tag BODY):

1. ‹script src="https://code.jquery.com/jquery-3.3.1.slim.min.js" integrity="sha384-q8i/X+965DzO0rT7abK41JStQIAqVgRVzpbzo5smXKp4YfRvH+8abtTE1Pi6jizo" crossorigin="anonymous"›
2. ‹script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.3/umd/popper.min.js" integrity="sha384-ZMP7rVo3mIykV+2+9J3UJ46jBk0WLaUAdn689aCwoqbBJiSnjAK/l8WvCWPIPm49" crossorigin="anonymous"›
